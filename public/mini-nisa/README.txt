Place the static demo assets for Mini NISA here:

1) CSV file for the CSV tab
   - Filename: sample.csv (or set env MINI_NISA_ASSET_CSV_PATH to custom path)

2) Image file for the Notes Image tab
   - Filename: observation-notes.png (or set env MINI_NISA_ASSET_IMAGE_PATH)

These files are preattached automatically on the first user message per conversation for the CSV and Image modes. No user uploads are allowed in the embed widget.
